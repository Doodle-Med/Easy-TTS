About the Easy Text-to-Speech Playground

Welcome to the Easy Text-to-Speech (TTS) Playground! This tool provides a simple interface to experiment with Google Cloud's Text-to-Speech API directly from your browser.

Features

  • Synthesize text into speech using various Google Cloud voices.
  • Adjust voice pitch and speaking rate.
  • Load available voices dynamically.
  • Playback synthesized audio with word highlighting.
  • Download the generated audio as an MP3 file.
  • Uses Google Sign-In (OAuth 2.0) to securely authorize API calls under your own Google Cloud account.

Purpose

This application is primarily intended as a personal tool and demonstration project for interacting with the Google Cloud TTS API via client-side JavaScript and OAuth 2.0 authentication.
